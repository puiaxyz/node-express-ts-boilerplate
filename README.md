# Express Typescript Boilerplate

Express + TypeScript API boilerplate with Drizzle ORM (PostgreSQL).

## Setup

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Environment**

   Copy `.env.example` to `.env` and set your values:

   ```bash
   cp .env.example .env
   ```

3. **Run**

   ```bash
   npm run dev
   ```

   Or build and start:

   ```bash
   npm run build
   npm start
   ```
